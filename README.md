# Browndog's More Ore Mod
A Minecraft mod that add more ores to the game! Made in MCreator.
# Ores

- Titanium
- Aetherium
- Sliver
- Steel
- Nickel
- Ruby
- Sapphire
- Topaz
- Opal
- Tanzanite
- Luminous Gem
- Zinc
- Lithium
- Tin
- Platinum
- Red Diamond
- Orange Diamond
- Yellow Diamond
- Green Diamond
- Blue Diamond
- Purple Diamond
- Black Diamond

**All ores include armor and pickaxes**
